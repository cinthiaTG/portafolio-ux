Case study: TrackOn

TrackOn es una plataforma digital diseñada para transformar y optimizar el proceso de mudanzas y traslados de productos, abarcando tanto a pequeñas y grandes empresas como a usuarios particulares que buscan un servicio de transporte confiable y eficiente.

La solución nace como respuesta a las limitaciones del modelo tradicional, ofreciendo una herramienta centralizada que permite gestionar envíos de forma ágil, segura y accesible, mejorando así la experiencia del usuario y eliminando las principales ineficiencias del sector.

Entender a los usuarios:

Llevar a cabo estudios de usuarios para comprender sus necesidades y comportamientos.

La investigación revela que los usuarios tienen dificultades para encontrar un servicio de envíos que garantice seguridad y transparencia en la localización de sus pedidos.

Paper wireframe

Para estructurar la solución, se creó un esquema inicial del flujo del usuario y los módulos clave del sistema.

Este mapa sirvió como base para definir jerarquías de información, prioridades y navegación, asi como a establecer el diseño básico de una pantalla..

Wireframes Digitales

Después de validar los flujos de navegación, se desarrollaron wireframes digitales en Figma, tanto para versión desktop como móvil. Estos prototipos permitieron visualizar la interfaz, jerarquía visual y funciones clave.

Versión Desktop

Wireframe para escritorio con diseño funcional y jerarquía clara para facilitar la gestión de envíos.

Versión Móvil

Wireframe optimizado para dispositivos móviles, asegurando usabilidad en pantallas pequeñas.
